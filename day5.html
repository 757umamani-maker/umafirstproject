HTML Form

  1. Whenever you want to create form you have to use form tag.
  2. Whenever you want to provide a box to user to fill information it is created by using input tag.
  3. And you have to mention type inside input tag that what kind of data you want to enter.
  4. To inform user that what he should fill in that boxes you have to use label tag.
  5. Whatever information user filled in form, it stored in backend in the form of name=value.
  6. So you have to add name keyword in input so that backend can understand that how these values will store.
  7. Many time you have noticed that in form you have option in question and you have to choose one it is done with the help of radio type.
  8. for and id is same in label and input respectively.They are used to connect label and input.
  9. In radio type you have to enter value attribute otherwise the value stored in backend will be on or off by default.
  10. Also in form there are questions in which you have to select more than one option it is done with the help of checkboxes type.In this also you have to give value attribute.
  11. Placeholder attribute is used so that you give hint to user that in which way he should enter information.
  12. You have input type like date,file,email,tel etc.
  13. Textarea is used when you have to enter many information in one box.It is not a normal text.
  14. For dropdown you use select tag.You have to give option inside select tag.And in option tag you should use value attribute.
  15. Required attribute is used for important information so that the form is not submit without taking it.
