What Is HTML?

HTML is a Hyper Text Markup Language.

  • It is called Hyper Text because it contains Hyper Links with whose help we can jump from one webpage to another.
  • And it is called Markup Language because it provide a symmetric meaning and proper structure to content which we write.

When we simply start writing a code then when we enter in next line and whatever we write we expect that in same way in which we write code the same will be shown in result.

But it is not reality when we press enter in our code for next line the same thing is not happening in output.

Also all the lines are in same font whether it is heading or a simple lines.

So for heading we use h1 tag inside< >.

Also for next line we use p inside< >.

Types of Heading Tags:

  1. H1 inside< >
  2. H2 inside< >
  3. H3 inside< >
  4. H4 inside< >
  5. H5 inside< >
  6. H6 inside< >

Priority order is:

H1>H2>H3>H4>H5>H6

There is only one H1 heading in code because it shows the topic of whole webpage while other heading tags(H2,H3,H4,H5,H6) can be used multiple times.

Some Points Regarding HTML

  • H1,H2,H3,H4,H5,H6,p are tags.
  • HTML element is from starting of H1 to end of H1 same with p tag.
  • For entering a line in between we can use hr tag inside< >.
  • For entering more space between two paragraphs we can use br tag inside< >.
  • For comment anything we use ctrl /.
  • Whatever we write inside tag is called attribute.

For making list,we have two types of list:

  1. Ordered List
  2. Unordered list
  • For ordered list we can use ol lag inside< >.
  • Same for unordered list we can use ul tag inside< >.
  • For entering items in list we use li tag inside< >.

How to enter Hyperlinks?

  • For entering hyperlinks we use anchor tag.
  • Inside it we have to write href to provide link where we want to reach.
  • And in between of anchor tag we provide the name or content so by clicking on it we can reach that link.
  • To open that link in new tab not in same tab we have to write target="_blank".

How to enter an image in website?

  • For entering an image we use img tag.
  • You have to enter src(source) from where you are takin photo.
  • Also you have to enter alt equals to something which is shown when image is not visible.
  • You can also enter height and width of image.
