  3. For making nested list you have to enter the type of list you want to make, inside li tag.
  4. Inside li tag you can use heading tag for making list with heading.

Table

  • For Table Creation we have to use table tag to declare that we are going to make table.
  • For introducing lines in your table you have to use border="1" inside table tag.
  • To declare a row you have to use tr tag inside< >.
  • For entering data we have to use td tag inside< >.
  • For the heading of column you have to use th(table heading) tag inside< >.
  • Components of Table:
    1. Table Header-For head of table we use thead tag inside< >.
    2. Table Body-For body of table we use tbody tag inside< >.
    3. Table Footer-For footer of table we use tfoot tag inside< >.
  • You can also use colspan and rowspan to extent column and row.
