How Data Transfer Between Two Computers?

In whatshapp when we chat with anyone how the message we sent is received by other person.

This is done with the help of IP(internet protocol) address.We thought that only message was sent but the reality is that not only message but the IP address of sender and receiver was also sent.How it was sent is shown below:

Sender IP address Message Receiver IP address

What Is IP address?

It is temporary address provide by ISP(internet service provider) when any device connected to internet.

If any user switch off internet for sometime and reconnect to internet after sometime then the IP address maybe same as earlier or maybe different.

What is DNS?

It is the book which have IP address of all the websites.

So whatever website for which you send request,it first comes to DNS(Domain Name System) and DNS gave your computer the IP address of that website.

Now your computer have the receiver IP address so it send request to that server.

In Depth of DNS

DNS is made with combination of:

  • Recursive server
  • Root server (There are total 13 root server globally)
  • TLD(Top Level Domain) server
  • Authoritative server

Now suppose you are searching for any reel in instagram and you send the request to instagram and after that you disconnect from internet for sometime.

When you reconnect to your internet it is not compulsory that ISP gave you the same IP address as before.Suppose you get the another IP address and you are waiting for your result but the result was sent to the IP address you have earlier.

And at present time the IP address you have earlier have been already given to any device (which is connected to internet at that time when you disconnect).So your result will show in other device.

If it is not a reel but a confidential data then you maybe in danger.So in this case we introduce MAC address.

What Is MAC address?

MAC(Medium Access Control) address is permanent physical address of your device.

So if your result was sent to IP address you have earlier the result will not open because the IP address is same but the MAC address is different.

In this way your confidential data is protected.So the request send from your computer to instagram server in this way:

Sender IP address Sender MAC address Request Receiver IP address Receiver MAC address

So if by chance you disconnected from internet then also you get your result because your device has permanent MAC address.

What is Port Numbers?

Suppose you get the data from server you want but you have opened so many apps so how the computer can understand that where the data is opened.

So here for rescue we use port number.Suppose you want your data to be opened at whatshapp so not only the IP and MAC address was sent but also the port number of place where you want to open it was also sent.

So now it is like this:

  • S-sender
  • R-receiver
S IP address S MAC address S Port number Data R IP address R MAC address R Port number

How Internet Works?

In older times we can transfer data through:

  1. Pendrive
  2. Cables

Both are not practical ways to transfer data in large scale.

So here we introduce LAN.

What is LAN?

LAN is a local area network where we connect many computers to a single switch.So whenever we want to transfer data from one computer to other then we sent it to switch and further switch sent it to that computer.So we didn't need many cables to connect many computers we simply connect them to single switch.

In this way only there are many Local Area Network all over the world.

Now suppose you want to transfer data from one computer in one LAN to other computer in another LAN so how it is possible?

For this we use Router.

Router is a device which connected to LAN and helps to transfer data from one LAN to other.

Every LAN has its own Router.And all Routers are connected globally.
